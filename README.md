# NeoVIM Config
## Introduction
- This nvim config uses vim-plug to install plugins.
- nvim is configured as an IDE.
- Easy LSP Installation
- Languages covered:
  - rust
  - javascript
  - typescript
  - java
- terminal plugin
- Dashboard startify: with a random saying
- nvim-tree for easy file handling
- telescope, neoclip
- lightspeed for quick jump to any location
- surround, multicursor, tagbar, commentary
- co pilot
- vimwiki, diary & taskwarrier integrations
